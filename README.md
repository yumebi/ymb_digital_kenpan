# YMB デジタル検版

Adobe Illustrator 用のデジタル検版(プリフライトチェック)スクリプトです。開いているドキュメントに対して印刷入稿前のチェックを一括実行し、**エラー / 警告 / 情報** の3分類+総合判定でレポートします。判定とレポートに特化した検査専用ツールで、オブジェクトの自動修正は行いません。

## 主な機能

- **25項目の自動チェック** — サイズ照合・塗り足し・トンボ / フォントのアウトライン化・文字セーフマージン / RGB混入・特色・インキ総量・リッチブラック / リンク切れ・画像カラーモード・実効解像度 / 透明効果・ヘアライン・オーバープリント・不要オブジェクト ほか
- **事前設定と連動した判定** — 仕上がりサイズ(A4/B5等+カスタム)と印刷カラー数(1色/4色/4色+特色/特色のみ)を指定すると、カラー系チェックの合否基準が連動
- **検出オブジェクトへジャンプ** — 結果一覧の行をダブルクリックで該当オブジェクトを選択してズーム(ロック/非表示レイヤーは自動解除)
- **原因と対応の解説付き** — 各検出に「なぜ問題か・どう直すか」の短い解説を表示(教育用途にも)
- **レポート出力** — HTML / CSV(UTF-8 BOM付き)で判定・検出一覧・設定値を保存
- **閾値のカスタマイズ** — 塗り足し幅・解像度閾値・インキ総量上限(標準300%/油性350%/UV380%プリセット)などを変更可、設定の保存/読込に対応
- **誤OKを出さない設計** — 自動判定できないケースは必ず「警告(要確認)」に倒します

## 使い方

1. チェックしたいドキュメントを Illustrator で開く
2. メニュー **[ファイル] > [スクリプト] > [その他のスクリプト...]**(Ctrl+F12 / Cmd+F12)で `DigitalKenpan.jsx` を選択
3. 設定ダイアログで仕上がりサイズ・印刷カラー数・閾値を指定して **「検版実行」**
4. 結果画面で総合判定と項目別の検出内容を確認(行ダブルクリックで該当箇所へジャンプ)
5. 必要に応じて「レポート保存」で HTML / CSV を書き出し

> 実行中は ESC キーの長押しで中断できます。設定は「設定を保存」でスクリプトと同じフォルダに保存され、次回起動時に復元されます。

## インストール

毎回メニューに表示したい場合は、`DigitalKenpan.jsx` を Illustrator のスクリプトフォルダにコピーして再起動します。

- Windows: `C:\Program Files\Adobe\Adobe Illustrator <バージョン>\Presets\ja_JP\スクリプト\`
- macOS: `/Applications/Adobe Illustrator <バージョン>/Presets/ja_JP/スクリプト/`

> `DigitalKenpan.jsx` は **UTF-8(BOM付き)** です。BOM無しで保存し直すと日本語が文字化けします。
> スクリプトフォルダに書き込み権限が無い環境では設定保存に失敗するため、その場合は書き込み可能なフォルダに置いて「その他のスクリプト...」から実行してください。

## 動作環境

- Adobe Illustrator(ExtendScript が動作するバージョン。CC 系で動作想定)
- Windows / macOS 両対応
- 1ファイル構成(`DigitalKenpan.jsx` のみで動作)

## 技術スタック

- ExtendScript(ES3相当)+ ScriptUI — 追加ランタイム不要、Illustrator 単体で動作
- チェックはレジストリ方式の独立モジュール構造(関数1個の追加で項目を拡張可能)
- ドキュメント走査は1回の再帰走査を全チェックで共有(大規模データ対応)
- リンク画像はファイルヘッダ自前解析(JPEG/PNG/TIFF/PSD/BMP/GIF、先頭64KBのみ読込)

詳細な判定ロジック・制限事項・チェック項目の追加方法は **[docs/SPEC.md](docs/SPEC.md)** を参照してください。
